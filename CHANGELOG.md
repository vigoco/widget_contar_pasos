# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-01-01

### Added
- Initial release of step_goal_widget package
- StepService for step tracking using pedometer plugin
- StepGoalViewModel for state management
- StepGoalWidget with circular progress indicator
- Compact and expanded view modes
- Tap gesture support to toggle views
- Unit tests for progress calculation
- Mock data fallback for unsupported platforms
- Customizable colors and text styles
- Full documentation and examples
